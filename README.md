# QA Test Interview

This project is a test, it is part of an interview process.
The project is based on Symfony 4, PHP (obviously) and Docker (for now).
Testing is done using PHPUnit which is provided as a Symfony package.

## Installation

Use the [composer](https://getcomposer.org) package manager to install the project, and then run while on the root of the project :

```bash
composer install
```

Once the dependencies are installed, use [docker](https://docker.com) to start the project with this command :

```bash
docker-compose up
```
And you're set ! Go check [localhost:8080](http://localhost:8080)
